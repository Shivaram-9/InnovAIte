# 🚀 InnovAIte – AI-Based Innovation Management System

## 📌 Introduction
InnovAIte is an AI-powered platform designed to manage and evaluate innovative ideas efficiently. It replaces manual processes with automated, intelligent workflows, enabling faster and smarter decision-making.

---

## ❗ Problem Statement
- Manual idea collection is inefficient  
- Slow evaluation process  
- Lack of proper analytics  
- Not scalable for large organizations  

---

## 🎯 Objective
- Automate idea submission  
- Evaluate ideas using AI  
- Rank ideas based on impact  
- Provide centralized management  

---

## 💡 Proposed Solution
InnovAIte is a web-based system where users can submit ideas, and the system evaluates them using AI-based scoring logic. Admins can view and analyze all ideas through a dashboard.

---

## 🛠 Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Database:** SQLite  
- **AI Logic:** Keyword-based scoring  

---

## ✨ Key Features
- Idea submission system  
- AI-based scoring  
- Admin dashboard  
- Real-time evaluation
